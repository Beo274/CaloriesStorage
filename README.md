# Сервис для обработки данных продуктах питания
## Репозиторий содержит код REST API, который осуществляет запись и вывод данных о продуктах и разделение этих продуктов на приемы пищи
> [!TIP]
> ## Установка
>- Клонировать репозиторий
>- Открыть папку с репозиторием в терминале
>- docker-compose --build
>- docker-compose up

>[!IMPORTANT]
>## Использование
>С помощью бразура или другого ПО, отправьте запрос на сервер
>### Endpoints
> POST /basket - добавить новую корзину (прием питания), формат:<br>
>- name
>- Список еды<br>
>GET /basket/{id} - получить корзину/прием пищи по идентификатору<br>
>POST /basket/{id}/add-food - добавить еду в конкретную корзину, формат:<br>
>- name
>- calories
>- proteins
>- fats
>- carbohydrates
>- mass
>- basket_id
>POST /del-food - удалить еду из списка
>- username<br>
>Запрос возвращает строку с именем удаленного продукта
